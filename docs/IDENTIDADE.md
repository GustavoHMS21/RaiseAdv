# Identidade — JusFlow

**Nome:** JusFlow
**Tagline:** "Seu escritório jurídico, em fluxo."
**Alternativas:** Causa, Lexy, Toga, Foro

## Posicionamento
Software jurídico moderno, simples e acessível para advogados autônomos e escritórios pequenos que não querem pagar R$ 200+ por mês em software legado.

## Valores visuais
- **Transparência / confiança** (essencial: dados jurídicos sensíveis)
- **Profissionalismo sóbrio** (não brincar com cores neon)
- **Moderno mas sério**

## Paleta
- Primary: `#1E3A8A` (azul marinho — autoridade, confiança)
- Accent: `#B8860B` (dourado sóbrio — tradição jurídica)
- Neutral: `#0F172A` (texto) / `#F8FAFC` (bg) / `#E2E8F0` (borders)
- Success: `#059669` / Danger: `#DC2626`

## Tipografia
- Headings: **Inter** (ou Geist) — limpa, profissional
- Body: **Inter**
- Mono (números/IDs processo): **JetBrains Mono**

## Tom de voz
- Direto, técnico quando preciso, sem jargão de vendas.
- Trata o advogado como profissional — não como "usuário".
